**Задача 1**

**Текст задачи**

Сформировать список из степеней числа 2.

**Алгоритм решения**

Программа запрашивает у пользователя целое число count. Если count больше нуля, вызывается функция items, которая генерирует список из count элементов, где каждый элемент является степенью двойки (от 2⁰ до 2ᶜᵒᵘⁿᵗ⁻¹), после чего список выводится на экран. Если введённое число меньше или равно нулю, программа выводит сообщение об ошибке. В конце программа возвращает 0.

**Задание 3**

**Задача 1**

Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать готовый класс Complex запрещено.

Алгоритм решения

Программа выполняет арифметические операции над двумя комплексными числами вида a + bi. Сначала она запрашивает у пользователя действительную (a) и мнимую (b) части для первого и второго комплексного числа. Затем последовательно вычисляет и выводит результаты четырёх операций: сумма вычисляется сложением соответствующих частей, разность — вычитанием соответствующих частей, произведение — по формуле (a₁a₂ - b₁b₂) + (a₁b₂ + b₁a₂)i, частное — по формуле (a₁a₂ + b₁b₂)/(a₂² + b₂²) + (b₁a₂ - a₁b₂)/(a₂² + b₂²)i с проверкой деления на ноль (когда оба a₂ и b₂ равны нулю). Результаты выводятся с форматированием: если мнимая часть положительная, используется знак "+", если отрицательная — знак "-", а при нулевой мнимой части выводится только действительная часть. В конце программа возвращает 0.
